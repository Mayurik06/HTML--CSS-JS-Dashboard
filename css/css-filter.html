CSS filter Property

  • The filter property defines visual effects (like blur and saturation) to an element (often <img>).
  • The filters that use percentage values (i.e. 75%), also accept the value as decimal (i.e. 0.75).

Filter: Blur

Syntax:

img {
filter: blur(5px);
}
