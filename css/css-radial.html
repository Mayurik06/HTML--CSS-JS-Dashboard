CSS Radial Gradients

  • A radial gradient is defined by its center.
  • To create a radial gradient you must also define at least two color stops.

Radial Gradient - Evenly Spaced Color Stops (this is default)

Syntax:

div.radial {
background-image: radial-gradient(#a688fa, white);
height: 200px;
}

Radial Gradient - Differently Spaced Color Stops

Syntax:

div.radial {
background-image: radial-gradient(#a688fa 5%, white 15%);
height: 200px;
}

Set Shape

The shape parameter defines the shape. It can take the value circle or ellipse. The default value is ellipse.

Syntax:

div.radial {
background-image: radial-gradient(circle, #1a1625, #a688fa, white);
height: 200px;
}

Example

This is an example of radial gradient(Set shape).

Use of Different Size Keywords

  • The size parameter defines the size of the gradient. It can take four values:

    • closest-side
    • farthest-side
    • closest-corner
    • farthest-corner

Syntax:

div.radial {
background-image: radial-gradient(closest-side at 60% 55%, #a688fa, white);
height: 200px;
}

This is an example of radial gradient(Use of Different Size Keywords).

Repeating a radial-gradient

The repeating-radial-gradient() function is used to repeat radial gradients.

Syntax:

div.radial {
background-image: radial-gradient(closest-side at 60% 55%, #a688fa, white);
height: 200px;
}
