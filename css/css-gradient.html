CSS Gradients

  • CSS gradients let you display smooth transitions between two or more specified colors.
  • CSS defines three types of gradients:

    • Linear Gradients (goes down/up/left/right/diagonally)
    • Radial Gradients (defined by their center)
    • Conic Gradients (rotated around a center point)

CSS Linear Gradients

To create a linear gradient you must define at least two color stops. Color stops are the colors you want to render smooth transitions among. You can also set a starting point and a direction (or an angle) along with the gradient effect.

Direction- Top to Bottom(this is default)

Example:

div.linear {
background-image: linear-gradient(#a688fa, white);
height: 200px;
}

This is an Example of Linear Gradient(top to bottom)

Direction - Left to Right

Example:

div.linear {
background-image: linear-gradient(to right, #a688fa, white);
height: 200px;
}

This is an Example of Linear Gradient(left to right)

Direction - Diagonal

You can make a gradient diagonally by specifying both the horizontal and vertical starting positions.

Example:

div.linear {
background-image: linear-gradient(to bottom, #a688fa, white);
height: 200px;
}

This is an Example of Linear Gradient(top to bottom)

Using Angles

If you want more control over the direction of the gradient, you can define an angle, instead of the predefined directions (to bottom, to top, to right, to left, to bottom right, etc.). A value of 0deg is equivalent to "to top". A value of 90deg is equivalent to "to right". A value of 180deg is equivalent to "to bottom".

Example:

div.linear {
background-image: linear-gradient(150deg, #a688fa, white);
height: 200px;
}

This is an Example of Linear Gradient(150deg)

Using Multiple colors stops

Example:

div.linear {
background-image: linear-gradient(#1a1625,#a688fa, white);
height: 200px;
}

This is an Example of Linear Gradient(using multiple colors)
