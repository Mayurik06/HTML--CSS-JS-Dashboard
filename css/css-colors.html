CSS Colors

  • Colors are specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values.

CSS Text Colors

You can set the color of text.

Example:

<h1 style="color: red;">This is heading</h1>
<p style="color: blue;">Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptate voluptatum odio perspiciatis quis tempore aliquam, officia sapiente necessitatibus facere architecto.</p>

Output:

This is heading

Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptate voluptatum odio perspiciatis quis tempore aliquam, officia sapiente necessitatibus facere architecto.

CSS Background Colors

You can set the background color for HTML elements.

Example:

<h1 style="background-color:DodgerBlue;">Hello World</h1>
<p style="background-color:Tomato;">Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptate voluptatum odio perspiciatis quis tempore aliquam, officia sapiente necessitatibus facere architecto.</p>

Output:

Hello World

Lorem ipsum dolor sit amet consectetur adipisicing elit. Et impedit provident aspernatur, laborum explicabo incidunt dolore, consequatur rerum ea vero in, odit accusamus ipsum.

CSS Border Colors

You can set the color of borders.

Example:

<h1 style="border:2px solid Tomato;">Hello World</h1>
<h1 style="border:2px solid DodgerBlue;">Hello World</h1>
<h1 style="border:2px solid Violet;">Hello World</h1>

Output:

Hello World

Hello World

Hello World
